# 日常步數記錄器

## 專案簡介
這是一個簡單的網頁計數工具，模擬記錄日常步數或任何需要計數的活動，讓使用者可以透過按鈕增加或減少數字，類似於一個輕量化的計數器。現已升級為PWA版本，支援真實自動計步功能！
